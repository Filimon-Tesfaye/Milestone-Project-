About This Project

This website was created as part of the IT3203 course project on web development and networking concepts. The goal is to demonstrate a responsive, mobile‑friendly site while explaining how HTTP has evolved and how it works.

Design Notes

  • A single external CSS file is used to keep styles consistent across all pages.
  • Media queries are used to adjust the layout for phones, tablets, and desktops.
  • Navigation, tables, and the quiz form are designed to remain usable on small screens.
  • Semantic HTML tags such as <header>, <main>, and <footer> improve accessibility and structure.
