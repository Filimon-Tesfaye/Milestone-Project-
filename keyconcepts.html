Key Concepts

This page explains 6 key concepts related to HTTP.

  1. Request/Response model — client sends requests, server issues responses.
  2. Statelessness — each HTTP transaction is independent.
  3. Headers — carry metadata (Content-Type, Cache-Control).
  4. Multiplexing — multiple streams on a single connection (HTTP/2 & QUIC).
  5. TLS / Encryption — security layer; QUIC integrates TLS 1.3.
  6. Caching — improves performance via Expires, Cache-Control, ETag.
